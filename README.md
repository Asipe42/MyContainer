# MyContainer

## 동적 배열 (Dynamic Array)
동적 배열은 크기가 고정되지 않고, 요소가 추가될 때마다 필요에 따라 크기가 자동으로 확장되는 배열입니다. 일반적으로 **배열 크기 증가**는 기존의 배열을 새로운 배열로 복사하는 방식으로 이루어집니다.

## 연결 리스트 (Linked List)
연결 리스트는 각 요소가 노드로 구성되며, 노드가 이전 또는 다음 노드를 참조하는 자료구조입니다. 노드는 데이터와 다음 노드를 가리키는 포인터(또는 링크)를 포함합니다.

## 스택 (Stack)
스택은 **후입선출(LIFO, Last In First Out)** 구조로, 가장 최근에 추가된 요소가 가장 먼저 제거되는 자료구조입니다. 주로 **push**와 **pop** 연산을 통해 데이터를 다룹니다.

## 큐 (Queue)
큐는 **선입선출(FIFO, First In First Out)** 구조로, 먼저 들어온 요소가 먼저 나가는 자료구조입니다.
